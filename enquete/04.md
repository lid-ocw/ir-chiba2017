- Q036.
  - SQlite文において、スペースは意味があるのでしょうか？
    - A. "SELECT * FROM table;" を "SELECT*FROMtable;"と書けないように，キーワード間には半角スペースが必要です．
ただし，見やすさのために入れるスペースや改行は無視されるため，次の2つのコマンドは等価です．

-----

     SELECT * FROM table;

-----

    SELECT  
      *  
    FROM  
      table  
    ;
